Op¶

class cvc5::api::Op¶

A cvc5 operator. An operator is a term that represents certain operators, instantiated with its required parameters, e.g., a term of kind BITVECTOR_EXTRACT.

bool operator==(const Op &t) const¶

Syntactic equality operator. Return true if both operators are syntactically identical. Both operators must belong to the same solver object.
