DatatypeSelector¶

class pycvc5.DatatypeSelector¶

A cvc5 datatype selector. Wrapper class for cvc5::api::DatatypeSelector.

getName()¶
Returns

the name of this datatype selector.

getRangeSort()¶
Returns

the range sort of this selector.

getSelectorTerm()¶
Returns

the selector opeartor of this datatype selector as a term.

getUpdaterTerm()¶
Returns

the updater opeartor of this datatype selector as a term.

Returns

True if this DatatypeSelector is a null object.
