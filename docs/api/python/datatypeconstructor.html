DatatypeConstructor¶

class pycvc5.DatatypeConstructor¶

A cvc5 datatype constructor. Wrapper class for cvc5::api::DatatypeConstructor.

getConstructorTerm()¶
Returns

the constructor operator as a term.

getName()¶
Returns

the name of the constructor.

getNumSelectors()¶
Returns

the number of selecters (so far) of this Datatype constructor.

getSelector()¶
Parameters

name – the name of the datatype selector.

Returns

the first datatype selector with the given name

getSelectorTerm()¶
Parameters

name – the name of the datatype selector.

Returns

a term representing the firstdatatype selector with the given name.

getSpecializedConstructorTerm()¶

Specialized method for parametric datatypes (see DatatypeConstructor::getSpecializedConstructorTerm()).

Parameters

retSort – the desired return sort of the constructor

Returns

the constructor operator as a term.

getTesterTerm()¶
Returns

the tester operator that is related to this constructor, as a term.

isNull()¶
Returns

True if this DatatypeConstructor is a null object.
