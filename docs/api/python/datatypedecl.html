class pycvc5.DatatypeDecl¶

A cvc5 datatype declaration. Wrapper class for cvc5::api::DatatypeDecl.

addConstructor()¶

Add a datatype constructor declaration.

Parameters

ctor – the datatype constructor declaration to add.

getName()¶
Returns

the name of this datatype declaration.

getNumConstructors()¶
Returns

number of constructors (so far) for this datatype declaration.

isNull()¶
Returns

True if this DatatypeDecl is a null object.

Returns

is this datatype declaration parametric?
