Datatype¶

class pycvc5.Datatype¶

Wrapper class for cvc5::api::Datatype.

__getitem__()¶

Return a constructor by index or by name.

getConstructor(self, unicode name)¶

Return a constructor by name.

getConstructorTerm(self, unicode name)¶
Returns

the term representing the datatype constructor with the given name (see Datatype::getConstructorTerm()).

getNumConstructors(self)¶
Returns

number of constructors.

getSelector(self, unicode name)¶

Return a selector by name.

hasNestedRecursion(self)¶
Returns

whether this datatype has nested recursion (see Datatype::hasNestedRecursion()).

isCodatatype(self)¶
Returns

whether this datatype corresponds to a co-datatype.

isFinite(self)¶
Returns

whether this datatype is finite.

isParametric(self)¶
Returns

whether this datatype is parametric.

isRecord(self)¶
Returns

whether this datatype corresponds to a record.

isTuple(self)¶
Returns

whether this datatype corresponds to a tuple.

isWellFounded(self)¶
Returns

whether this datatype is well-founded (see Datatype::isWellFounded()).
